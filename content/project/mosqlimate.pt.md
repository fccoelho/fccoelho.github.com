## Sobre

O Mosqlimate é uma plataforma de ciência de dados projetada para produzir previsões utilizáveis do burden de dengue, incorporando todas as fontes de dados disponíveis com atenção especial ao impacto das tendências climáticas futuras.

## Objetivos

- Desenvolver modelos de previsão para a incidência de dengue
- Integrar dados climáticos e epidemiológicos
- Garantir a reprodutibilidade de todo o pipeline de previsão
- Rastrear dados, código-fonte dos modelos e previsões

## Financiamento

Este projeto é financiado pela **Wellcome Trust**.
